# UworkPlus Low-code Editor

Visual drag-and-drop page builder used with the [uworkplus-code](../uworkplus-code/) VS Code extension.

## Development

```bash
cd uworkplus-code-lowcodeEditor
yarn install
yarn dev
```

Deploy under `/lowcodeEditor/` on your static host (see `VITE_BASE_URL` in `.env.*`).

## Configuration

Copy `env.example` to `.env.local` and set API endpoints for your backend.
